# Analyze — Evidence-Driven Investigation

Use for ambiguous, causal, evidence-heavy questions.

## Core Contract
1. **Observation** — what was actually observed
2. **Hypotheses** — competing explanations
3. **Evidence For/Against** — ranked evidence
4. **Current Best Explanation**
5. **Critical Unknown**
6. **Recommended Next Step**

## Evidence Hierarchy
1. Controlled reproductions
2. Primary source artifacts
3. Multiple converging sources
4. Single-source inference
5. Weak circumstantial clues
6. Intuition/speculation

## Output Format
### Observed Result
### Ranked Hypotheses
| Rank | Hypothesis | Confidence | Evidence Strength |
### Evidence Summary
### Most Likely Explanation
### Critical Unknown
### Recommended Next Step
